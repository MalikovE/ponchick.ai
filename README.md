# ponchick.ai
Recognize is it ponchick or not  

Examples:  

Input grid:  
\_X\_  
\_X\_   
XX\_   
It's not a ponchik!  
  
Input grid:    
\_XX   
X\_X  
XXX  
It's a ponchik!  
  

Input grid:  
\_X\_  
\_X\_  
X\_\_  
It's not a ponchik!  
  
Input grid:  
\_X\_  
X\_X  
\_X\_  
It's a ponchik!
